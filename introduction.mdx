## Fluxo basico de integracao

```mermaid
sequenceDiagram
    participant App as Sua Aplicacao
    participant API as Real Oficial API
    participant Worker as Workers

    App->>API: POST /login (credenciais)
    API-->>App: Token de autenticacao

    App->>API: POST /createShorts (URL do video)
    API-->>App: Projeto criado

    Worker->>Worker: Processa video e gera cortes

    App->>API: GET /shorts/{projectId}
    API-->>App: Lista de cortes gerados

    App->>API: POST /shorts/{projectId}/{shortId}/render
    API-->>App: Renderizacao iniciada

    App->>API: GET /renders
    API-->>App: Status e URL do video renderizado
```

## Rate Limits

A API possui limites de requisicoes para garantir estabilidade:

| Endpoint | Limite |
|----------|--------|
| Login | 60 requisicoes/minuto |
| Criar Projeto | 10 requisicoes/minuto |
| Renderizar | 30 cortes/hora |

---

## Endpoint

```
GET https://api.realoficial.com.br/api/v1/projects
```

<Note>
Este endpoint requer autenticacao via Bearer Token.
</Note>

## Query Parameters

<ParamField query="page" type="integer">
  Numero da pagina (padrao: 1)
</ParamField>

<ParamField query="per_page" type="integer">
  Quantidade de itens por pagina (1-100, padrao: 10)
</ParamField>

## Resposta de Sucesso

<ResponseField name="data" type="array">
  Lista de projetos
  <Expandable title="Propriedades do projeto">
    <ResponseField name="id" type="string">
      ID unico do projeto (ULID)
    </ResponseField>
    <ResponseField name="title" type="string">
      Titulo do projeto
    </ResponseField>
    <ResponseField name="url" type="string">
      URL do video original
    </ResponseField>
    <ResponseField name="status" type="string">
      Status atual do projeto
    </ResponseField>
    <ResponseField name="goal" type="string">
      Tipo: `shorts`, `reframe` ou `empty`
    </ResponseField>
    <ResponseField name="video_layout" type="string">
      Layout: `single`, `split`, `react`, `gaming`
    </ResponseField>
    <ResponseField name="aspect_ratio" type="string">
      Formato: `vertical` ou `horizontal`
    </ResponseField>
    <ResponseField name="clip_duration" type="string">
      Duracao alvo dos cortes
    </ResponseField>
    <ResponseField name="start_time" type="number">
      Tempo inicial processado
    </ResponseField>
    <ResponseField name="end_time" type="number">
      Tempo final processado
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Data de criacao (ISO 8601)
    </ResponseField>
    <ResponseField name="thumbnail_url" type="string">
      URL da thumbnail do video
    </ResponseField>
    <ResponseField name="metadata" type="object">
      Metadados do video original
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="links" type="object">
  Links de navegacao da paginacao
</ResponseField>

<ResponseField name="meta" type="object">
  Metadados da paginacao
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.realoficial.com.br/api/v1/projects?page=1&per_page=20" \
  -H "Authorization: Bearer seu_token_aqui" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.realoficial.com.br/api/v1/projects?page=1&per_page=20', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer seu_token_aqui',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data.data); // Array de projetos
console.log(data.meta.total); // Total de projetos
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer seu_token_aqui',
    'Content-Type': 'application/json'
}

params = {
    'page': 1,
    'per_page': 20
}

response = requests.get(
    'https://api.realoficial.com.br/api/v1/projects',
    headers=headers,
    params=params
)

data = response.json()
projects = data['data']
```

```php PHP
$ch = curl_init();

curl_setopt_array($ch, [
    CURLOPT_URL => 'https://api.realoficial.com.br/api/v1/projects?page=1&per_page=20',
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer seu_token_aqui',
        'Content-Type: application/json'
    ]
]);

$response = curl_exec($ch);
$data = json_decode($response, true);
$projects = $data['data'];
```

## Status dos Projetos

<CardGroup cols={2}>
  <Card title="Em Processamento" icon="spinner">
    - `pending` - Aguardando
    - `downloading` - Baixando
    - `transcribing` - Transcrevendo
    - `transcribed` - Transcrito
    - `shorts_created` - Cortes criados
  </Card>
  <Card title="Concluido" icon="check">
    - `face_analysis_done` - Pronto para uso
    - `refunded` - Reembolsado
  </Card>
</CardGroup>

## Filtrando por Status

Para filtrar projetos prontos, verifique o campo `status`:

```javascript
const readyProjects = projects.filter(p => p.status === 'face_analysis_done');
const processingProjects = projects.filter(p =>
  ['pending', 'downloading', 'transcribing'].includes(p.status)
);
```

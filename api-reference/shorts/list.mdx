---
title: 'Listar Shorts'
api: 'GET /shorts/{projectId}'
description: 'Liste todos os cortes (shorts) de um projeto'
---

## Endpoint

```
GET https://api.realoficial.com.br/api/v1/shorts/{projectId}
```

<Note>
Este endpoint requer autenticacao via Bearer Token.
</Note>

## Path Parameters

<ParamField path="projectId" type="string" required>
  ID do projeto (ULID)
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Numero da pagina (padrao: 1)
</ParamField>

<ParamField query="limit" type="integer">
  Quantidade de itens por pagina (1-100, padrao: 20)
</ParamField>

<ParamField query="sort" type="string">
  Ordenacao dos resultados:
  - `score` (padrao) - Por pontuacao de relevancia
  - `time` - Por tempo de inicio no video
  - `duration` - Por duracao do corte
</ParamField>

<ParamField query="render" type="string">
  Filtrar por status de renderizacao:
  - `all` (padrao) - Todos os cortes
  - `rendered` - Apenas renderizados
  - `not_rendered` - Apenas nao renderizados
  - `liked` - Apenas marcados como favoritos
</ParamField>

## Resposta de Sucesso

<ResponseField name="message" type="string">
  Mensagem de confirmacao
</ResponseField>

<ResponseField name="project" type="object">
  Dados resumidos do projeto
</ResponseField>

<ResponseField name="data" type="object">
  Objeto paginado com os shorts
  <Expandable title="Propriedades do short">
    <ResponseField name="id" type="string">
      ID unico do short (ULID)
    </ResponseField>
    <ResponseField name="title" type="string">
      Titulo do corte
    </ResponseField>
    <ResponseField name="start_time" type="number">
      Tempo de inicio no video original (segundos)
    </ResponseField>
    <ResponseField name="end_time" type="number">
      Tempo de fim no video original (segundos)
    </ResponseField>
    <ResponseField name="score" type="number">
      Pontuacao de relevancia (0-100)
    </ResponseField>
    <ResponseField name="status" type="string">
      Status do processamento
    </ResponseField>
    <ResponseField name="has_render" type="boolean">
      Se ja foi renderizado
    </ResponseField>
    <ResponseField name="feedback" type="string">
      Feedback do usuario (`like`, `dislike`, ou array de motivos)
    </ResponseField>
    <ResponseField name="timeline" type="object">
      Dados da timeline para edicao (assets, tracks, items)
    </ResponseField>
    <ResponseField name="renders" type="array">
      Lista de renderizacoes deste short
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Metadados agregados
  <Expandable title="Propriedades">
    <ResponseField name="average_score" type="number">
      Pontuacao media dos cortes
    </ResponseField>
    <ResponseField name="average_duration" type="number">
      Duracao media em segundos
    </ResponseField>
  </Expandable>
</ResponseField>

## Entendendo a Pontuacao (Score)

A pontuacao indica a relevancia do corte identificado pela IA:

| Faixa | Classificacao | Descricao |
|-------|---------------|-----------|
| 80-100 | Excelente | Momentos de alto engajamento potencial |
| 60-79 | Bom | Bons trechos para conteudo |
| 40-59 | Medio | Pode funcionar com edicao |
| 0-39 | Baixo | Menos recomendado |

<Info>
**Dica**: Comece renderizando os cortes com maior pontuacao para maximizar o engajamento.
</Info>

## Obter Short Especifico

Para obter detalhes de um short especifico:

```
GET /shorts/{projectId}/{shortId}
```

<RequestExample>
```bash cURL
curl -X GET "https://api.realoficial.com.br/api/v1/shorts/PROJECT_ID/SHORT_ID" \
  -H "Authorization: Bearer seu_token_aqui"
```
</RequestExample>

Retorna o objeto completo do short incluindo a timeline para edicao.

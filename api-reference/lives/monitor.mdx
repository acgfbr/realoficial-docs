## Endpoint

```
POST https://api.realoficial.com.br/api/v1/lives/monitor
```

<Note>
Este endpoint requer autenticacao via Bearer Token.
</Note>

Inicia o monitoramento de uma live na Twitch, Kick ou YouTube. Se a live ja estiver sendo monitorada por outro usuario, o sistema reutiliza o mesmo registro — nao cria duplicata.

## Body Parameters

<ParamField body="name" type="string" required>
  Nome do monitor (max 255 caracteres)
</ParamField>

<ParamField body="url" type="string" required>
  URL da live (Twitch, Kick ou YouTube)
</ParamField>

<ParamField body="prompt" type="string">
  Prompt customizado (max 2000 caracteres)
</ParamField>

<ParamField body="visibility" type="string" required>
  Visibilidade do monitor: `public` ou `private`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.realoficial.com.br/api/v1/lives/monitor" \
  -H "Authorization: Bearer seu_token_aqui" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Gaules ao vivo",
    "url": "https://www.twitch.tv/gaules",
    "prompt": "Foque em momentos engraçados",
    "visibility": "public"
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://api.realoficial.com.br/api/v1/lives/monitor',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer seu_token_aqui',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Gaules ao vivo',
      url: 'https://www.twitch.tv/gaules',
      prompt: 'Foque em momentos engraçados',
      visibility: 'public'
    })
  }
);

const data = await response.json();
console.log(data.monitor);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer seu_token_aqui',
    'Content-Type': 'application/json'
}

payload = {
    'name': 'Gaules ao vivo',
    'url': 'https://www.twitch.tv/gaules',
    'prompt': 'Foque em momentos engraçados',
    'visibility': 'public'
}

response = requests.post(
    'https://api.realoficial.com.br/api/v1/lives/monitor',
    headers=headers,
    json=payload
)

data = response.json()
```

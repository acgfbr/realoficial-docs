---

## Endpoint

```
GET https://api.realoficial.com.br/api/v1/lives/monitors
```

<Note>
Este endpoint requer autenticacao via Bearer Token.
</Note>

Retorna os monitors do usuario autenticado, paginados (20 por pagina).

## Query Parameters

<ParamField query="visibility" type="string">
  Filtrar por visibilidade: `public` ou `private`
</ParamField>

<ParamField query="page" type="integer">
  Numero da pagina (padrao: 1)
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.realoficial.com.br/api/v1/lives/monitors?visibility=public" \
  -H "Authorization: Bearer seu_token_aqui"
```
</RequestExample>
